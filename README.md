# Heart Disease Analysis and Prediction

## Description
This project analyzes a heart disease dataset and uses Machine Learning techniques to predict whether a patient is likely to have heart disease based on medical attributes such as age, cholesterol, blood pressure, and other health indicators.

## Tools & Technologies
- Python (pandas, numpy, matplotlib)
- Machine Learning (scikit-learn)

## Dataset
Dataset is available in the `data/` folder.

## Objective
To analyze health data and build a Machine Learning model to predict the presence of heart disease.

## Steps Performed
- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Feature selection  
- Model building using classification algorithm  
- Model evaluation  

## Key Insights
- Age and cholesterol are major factors influencing heart disease  
- Certain medical indicators show strong correlation with disease presence  
- Patterns observed between patient attributes and outcomes  

## Machine Learning
- Model Used: Logistic Regression / Decision Tree  
- Built classification model to predict heart disease  
- Evaluated model performance using accuracy metrics  

## Final Result
- Successfully built a predictive model for heart disease  
- Model can classify patients based on risk factors  
- Visualized important health trends  

## Conclusion
This project demonstrates how Machine Learning can be used in healthcare to assist in early detection and risk analysis of heart disease.
